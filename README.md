# Spark

A Trinket.io-style platform for students to build, preview, and publish HTML sites.

## Features

- Simple name-based sign in
- Create and manage coding projects
- File tree with HTML, CSS, JS, and more
- Live preview while editing
- Publish sites to shareable URLs at `/p/[slug]`

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js 16 (App Router)
- Tailwind CSS 4
- Bootstrap Icons
- File-based storage in `data/projects/`
